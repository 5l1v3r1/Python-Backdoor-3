Remote Access Trojan built in Python using the socket module to open a connection with the client on the target computer to gain unauthorized access.
Operating System: Windows

[Capabilities]

- Send Message (VBS)
- View System Information
- Shutdown Computer
- Restart Computer
- Lock Computer
- Write Code on target machine
- Activate Webcam
- Screenshot
- Get RAT Directory
- View Files
- Append Connection (Keep it running while not connected)
- Close Connection (deletes program and any other logs)

This program when ran automatically saves to targets startup folder
