Remote Access Trojan built in Python using the socket module to open a connection with the client on the target computer to gain unauthorized access.
Operating System: Windows

[Capabilities]

- Send Message (VBS)
- View System Information
- Shutdown Computer
- Restart Computer
- Lock Computer
- Activate Webcam
- Screenshot
- Get RAT Directory
- View Files
- Check Connection
- Append Connection (Keeps program running while not connected)
- Close Connection (terminates connection and clears logs)

This program when ran automatically saves to targets startup folder
